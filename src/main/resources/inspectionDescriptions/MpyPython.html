Detects when a valid Python 3.10+ interpreter is not configured for MicroPython tooling.

This inspection ensures that the required Python interpreter is available and properly configured. The MicroPython Tools plugin relies on a local Python 3.10+ interpreter to install and manage stub packages using pip. Without a valid interpreter, stub installation and background tooling may fail silently.
